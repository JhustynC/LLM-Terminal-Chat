# LLM-Terminal-Chat

make for: Jhustyn Carvajal with love 

## Install dependencies

```
npm i 
```

## Setup env variables 

- Create a .env in the root of project
- Put the .env.template variables in the new .env 
- You need a API_KEY for Google Gemini 

## Up the project


For Typescript:
```
npm run dev 
```

For javascript
```
npm i dev:js 
```
